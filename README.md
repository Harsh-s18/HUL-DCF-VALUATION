## 🔧 Methodology

- 8-year historical financial statement analysis (FY18–FY26)
- Free Cash Flow to Firm (FCFF) based DCF with mid-year convention
- Blume-adjusted Beta via OLS regression (105 weekly observations vs Nifty 50)
- WACC derived through peer comparable beta unlevering & relevering
- Terminal Value using Gordon Growth Model (g = 4%)
- Sensitivity analysis across WACC (9.5%–11.25%) & growth rate (4%–6%)
- Comparable company analysis across 9 Indian FMCG peers
- Football field valuation summary (Bear to Bull range)
